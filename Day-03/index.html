Array Methods: map, filter, reduce

map transforms every element, filter keeps elements that match a condition, and reduce combines values into one.

// map
const nums = [1,2,3,4];
const doubled = nums.map(n => n * 2); // [2,4,6,8]

// filter
const evens = nums.filter(n => n % 2 === 0); // [2,4]

// reduce
const sum = nums.reduce((acc, n) => acc + n, 0); // 10
